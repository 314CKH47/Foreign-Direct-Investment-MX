Before to start:
1-Open terminal in folder «Dashboard». You can enter using cd 
1.1-Activate the venv (virtual enviroment):
1.2-Windows: dash\Scripts\activate
1.2-Linux/macOS: dash\bin\activate 
2-pip install -r requirements.txt
***Opcional***
2.1-Write: 'cls' to clear terminal (Windows)
2.1-Write: 'clear' to clear terminal (Linux/macOS)
2.1-Ctrl + L to clear terminal (macOS)
****------***
3-Command in terminal: python Dashboard.py (Windows)
3-Command in terminal: python3 Dashboard.py (Linux)
4-Ctrl + Click over «Dash is running on http://127.0.0.1:80/»
or copy the link and brows it.
